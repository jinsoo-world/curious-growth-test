# 큐리어스 성장유형 진단검사

큐리어스 브랜드 스타일의 성장유형 진단검사 웹사이트입니다.

## 기능

- 📋 15개의 질문으로 성장유형 진단
- 🎯 4가지 성장유형 (A, B, C, D) 판정
- 📊 실시간 진행 상황 표시
- 🎨 큐리어스 브랜드 스타일 UI/UX (녹색 강조, 깔끔한 디자인)
- 📱 반응형 디자인
- 👤 사용자 정보 수집 (이름, 휴대폰번호)
- 💾 진단 데이터 저장 (로컬 스토리지)
- 🔐 관리자 페이지 (비밀번호 보호)
- 📥 CSV 내보내기 기능

## 성장유형

- **A. 야망 있는 크리에이터**: 만들고, 드러내고, 확장하려는 유형
- **B. 결정형 리더**: 시간보다 결과를 중시하는 투자형
- **C. 성장 탐색자**: 불안·눈치·준비 부족 상태
- **D. 조용한 실력자**: 역량은 있으나 표현·노출을 꺼리는 유형

## 설치 및 실행

```bash
cd apps/curious-growth-test
npm install
npm run dev
```

개발 서버가 `http://localhost:3002`에서 실행됩니다.

## 주요 특징

1. **진행 단계 시각화**: 3단계 프로세스를 카드 형태로 표시
2. **실시간 진행 바**: 현재 진행 상황을 시각적으로 표시
3. **질문 인디케이터**: 모든 질문의 답변 상태를 한눈에 확인
4. **유형별 상세 결과**: 현재 상태, 고민, 필요한 것, 추천 프로그램 제공
5. **데이터 수집**: 사용자 이름, 휴대폰번호, 문항별 답변, 결과 유형 저장
6. **관리자 기능**: 비밀번호로 보호된 관리자 페이지에서 모든 데이터 확인 및 CSV 내보내기

## 관리자 접근

1. 시작 페이지 하단의 "관리자" 버튼 클릭
2. 비밀번호 입력: `curious2024` (기본값)
3. 저장된 모든 진단 데이터 확인 가능
4. CSV 파일로 내보내기 가능
5. 전체 데이터 삭제 가능

## 데이터 저장 형식

각 진단 결과는 다음 정보를 포함합니다:
- 사용자 이름
- 휴대폰번호
- 진단 일시
- 문항별 답변 (질문 ID, 질문 텍스트, 선택한 답변)
- 결과 유형 (A, B, C, D)
- 각 유형별 점수
